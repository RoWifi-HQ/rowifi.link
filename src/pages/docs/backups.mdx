# Backups

Backups is a premium module that lets you backup your server configuration and import it into any servers you own.

## Creating a Backup

To create a backup, run the command `/backup new`. It requires only one argument: `name`.

</div>

## Importing a Backup

To import an existing backup into a server, run the command `/backup restore`. It requires only one argument: `name`. This is
the name you provided when you created the backup.
